## Version 0.7 Candidate

### Conversational Completion

Replace completion dialogs with Guide conversations.

Instead of displaying buttons in a MessageBox, Guide should ask:

"Everything is organized. What would you like to do next?"

Possible responses:

- Open the organized folder
- Show me the parent folder
- Review what changed
- Organize another folder
- Close

Rationale

The completion experience should mirror the beginning of the workflow:
Guide recommends, Friend decides.
